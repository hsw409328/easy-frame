# easy-frame

easy-frame 遵循composer psr4规范

#依赖

php >= 5.4

php-pdo 扩展
    
#运行

git clone https://github.com/hsw409328/easy-frame.git

cd easy-frame 

composer install
    
#ngingx

配置vhost

配置host

开启nginx rewrite
    

在server中添加：
  
    if (!-e $request_filename) { 
     rewrite ^(.*)$ /index.php?$1 last; break; 
    }

#注意：

    apps\config有两种环境的配置，dev为开发环境，pro为线上环境，请自行修改
